HTML (HyperText Markup Language)

Structure • Elements • Semantics • Accessibility

HTML is the foundation of the web. It provides the structure for web pages by defining elements such as headings, paragraphs, images, links, forms, and more. Understanding HTML is essential to create well-structured, accessible, and semantically meaningful websites.

Core topics include elements & attributes, semantic tags, headings & paragraphs, links & images, lists, tables, forms, multimedia embedding, and accessibility best practices .

Demonstration: HTML Structure

A simple HTML page structure demonstrating common elements. 

<!DOCTYPE html>
<html>
  <head><title>Demo</title></head>
  <body>
    <h1>Hello World</h1>
    <p>This is a paragraph.</p>
    <a href="#">Click Me</a>
  </body>
</html>

Key HTML Elements

Headings

<h1> to <h6> define headings for content structure.

Paragraphs

<p> defines a paragraph of text on the web page.

Links

<a> creates hyperlinks to other pages or resources.

Images

<img> embeds images in your HTML document.
